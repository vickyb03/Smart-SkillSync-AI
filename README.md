# Smart SkillSync AI 🚀

An AI-based skill matching system that analyzes candidate skills and compares them with job requirements using Machine Learning techniques.

---

## 🔹 Features
- Login & Dashboard UI
- Skill input and analysis
- ML-based matching system
- Missing skill detection
- Real-time result visualization

---

## 🔹 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Machine Learning: TF-IDF, Cosine Similarity
- Libraries: Scikit-learn, Pandas, NumPy

---

## 🔹 How It Works
1. User enters skills
2. System processes input
3. ML model compares with job requirements
4. Outputs:
   - Match score
   - Missing skills
